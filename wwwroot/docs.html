Documentation

Information

This Formula 1 Web API stores information about Formula 1 drivers. The information available is name, age, nationality, and an image of the driver.

Available endpoints

Base url: http://localhost:5282/api

How to use API endpoints

GET

{

Get all: http://localhost:5282/api/drivers

Get by id: http://localhost:5282/api/id

Get by name: http://localhost:5282/api/drivers/GetDriverByName/name

}

Driver schema

Key Type Description
id int The id of the driver
name string The name of the driver
age int The age of the driver
nationality string The country where the driver is from
image string A picture of the driver in PNG format
